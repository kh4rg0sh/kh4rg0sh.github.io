## Explanation

**Lemma.** For an increasing sequence of length $k$ consisting of positive integers that sum to $n$, we must have

$$
\frac{k(k+1)}{2} \le n
$$

The first observation is that if $d$ is the GCD of the $k$ integers, then $d$ must divide $n$.

Therefore, we can iterate over all divisors of $n$ and check whether there exists an increasing sequence of length $k$ with GCD equal to that divisor.

Using the lemma, such a sequence exists if

$$
\frac{k(k+1)}{2} \le \frac{n}{d}
$$

Hence, the maximum valid divisor $d$ gives the answer.

The construction for a valid divisor $d$ is straightforward:

$$
\left(
d,\,
2d,\,
3d,\,
\ldots,\,
(k-1)d,\,
n-\frac{k(k-1)}{2}d
\right)
$$

The first $k-1$ elements are the smallest possible distinct multiples of $d$. The final element absorbs the remaining sum while preserving the increasing order.

## Code

```cpp
void solve() {
    ll n, k;
    std::cin >> n >> k;

    ll ans = -1;

    if (k > 1e6) {
        std::cout << -1 << '\n';
        return;
    }

    if ((k * (k + 1)) / 2 > n) {
        std::cout << -1 << '\n';
        return;
    }

    for (ll i = 1; i * i <= n; i++) {
        if (n % i == 0) {
            if ((k * (k + 1)) / 2 <= (n / i)) {
                ans = std::max(ans, i);
            }

            if (i * i != n) {
                if ((k * (k + 1)) / 2 <= i) {
                    ans = std::max(ans, n / i);
                }
            }
        }
    }

    if (ans == -1) {
        std::cout << -1 << '\n';
    } else {
        ll sum = 0;

        for (ll i = 1; i < k; i++) {
            sum += i * ans;
            std::cout << i * ans << ' ';
        }

        std::cout << n - sum << '\n';
    }
}
```
