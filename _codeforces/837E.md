## Explanation

**Lemma.** For any two natural numbers $a$ and $b$,

$$
\gcd(a,b) \mid \left(b-k\gcd(a,b)\right)
$$

for every $k \in \mathbb{Z}^+$

However, it is not necessarily true that

$$
\gcd\left(a,\, b-k\gcd(a,b)\right)=\gcd(a,b)
$$

for all $k$.

Therefore, we need to determine when the GCD gains an additional factor. Equivalently, we seek the minimum $k$ such that

$$
\gcd\left(
\frac{a}{\gcd(a,b)},
\frac{b}{\gcd(a,b)}-k
\right) > 1
$$

Observe that

$$
\gcd\left(
\frac{a}{\gcd(a,b)},
\frac{b}{\gcd(a,b)}-k
\right)
\mid
\frac{a}{\gcd(a,b)}
$$

Hence, any extra factor must be a divisor of

$$
\frac{a}{\gcd(a,b)}
$$

We can therefore iterate over all divisors of

$$
\frac{a}{\gcd(a,b)}
$$

and determine the smallest valid value of $k$.

Enumerating divisors takes $\mathcal{O}(\sqrt{a})$ time, and the process repeats at most $\mathcal{O}(\log a)$ times because the value of $a$ decreases whenever a new factor is extracted.

Therefore, the overall complexity is

$$
\mathcal{O}(\sqrt{a}\log a).
$$

## Code

```cpp
void solve() {
    ll a, b;
    std::cin >> a >> b;

    ll ans = 0;

    while (b > 0) {
        ll d = std::gcd(a, b);

        a /= d;
        b /= d;

        if (a == 1) {
            ans += b;
            break;
        }

        ll best = LLONG_MAX;

        for (ll i = 1; i * i <= a; i++) {
            if (a % i == 0) {
                if (i > 1) {
                    best = std::min(best, b % i);
                }

                if (i * i != a) {
                    best = std::min(best, b % (a / i));
                }
            }
        }

        ans += best;
        b -= best;
    }

    std::cout << ans << '\n';
}
```
