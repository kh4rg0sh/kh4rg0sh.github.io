If D(nk) = kD(n), where D(n) = sum of digits of n, then each digit of n must become kn so we must have no carry overs else it wont satisfy => kd <= 9 which means, if d is a digit of n then d <= floor(9 / k). 

to count such number we just do ((9 / k) + 1)^r - ((9 / k) + 1)^l

```c++
void solve() {
    i64 l, r, k; cin >> l >> r >> k;
    z1 val = (9 / k) + 1;
    z1 up = binpow(val, r);
    z1 down = binpow(val, l);
    cout << up - down << '\n';
}   
```
