lowercase the entire string and compare against "yes"

```c++
void solve() {
    string s; cin >> s;
    std::transform(s.begin(), s.end(), s.begin(), [](unsigned char c) {
        return std::tolower(c);
    });
    cf(s == "yes");
}
```
