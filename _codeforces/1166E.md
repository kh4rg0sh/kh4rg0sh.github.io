## Explanation

**Lemma 1.** Consider two sets of natural numbers $A_1$ and $A_2$. If $A_1 \subseteq A_2$, then

$$
\operatorname{lcm}(A_1) \le \operatorname{lcm}(A_2).
$$

**Proof.** This follows immediately by induction on the size of the set.

**Lemma 2.** Suppose the sets of shop indices visited by Dora are

$$
D_1, D_2, \ldots, D_m.
$$

If the condition in the problem statement is true, then for any $i \ne j$ we must have

$$
D_i \cap D_j \neq \varnothing.
$$

**Proof.**

Let

$$
S_1, S_2, \ldots, S_m
$$

denote the sets of shop indices visited by Swiper.

For each day $i$, the sets $D_i$ and $S_i$ form disjoint partitions of the set of shops.

Suppose for contradiction that

$$
D_i \cap D_j = \varnothing.
$$

Then

$$
D_j \subseteq S_i.
$$

Using Lemma 1,

$$
\operatorname{lcm}(S_j)
<
\operatorname{lcm}(D_j)
\le
\operatorname{lcm}(S_i)
<
\operatorname{lcm}(D_i).
$$

Since

$$
D_j \subseteq S_i,
$$

we also have

$$
D_i \subseteq S_j,
$$

which implies

$$
\operatorname{lcm}(D_i)
\le
\operatorname{lcm}(S_j).
$$

This is a contradiction. Therefore,

$$
D_i \cap D_j \neq \varnothing
$$

for every pair of distinct days.

### Back to the Problem

We now show that the above condition is also sufficient.

Choose $m$ distinct primes

$$
p_1, p_2, \ldots, p_m.
$$

For each store $i$, define

$$
a_i = \prod_{j \in D} p_j,
$$

where $D$ is the set of days on which Dora visits the $i$-th store.

Now consider day $i$. Let $D_i$ be the set of stores visited by Dora and $S_i$ the set visited by Swiper.

The prime $p_i$ does not divide

$$
\operatorname{lcm}(S_i),
$$

because Dora does not visit any of those stores on day $i$.

However, for every $j$, the condition

$$
D_i \cap D_j \neq \varnothing
$$

guarantees that some store visited on day $i$ also contains the prime factor $p_j$. Therefore,

$$
p_j \mid \operatorname{lcm}(D_i)
$$

for all $j$.

Hence,

$$
\operatorname{lcm}(D_i)
>
\operatorname{lcm}(S_i),
$$

which proves that the pairwise-intersection condition is sufficient.

Therefore, the problem reduces to checking whether every pair of Dora's day-sets has a non-empty intersection.

void solve() {
    ll m, n;
    std::cin >> m >> n;

    std::vector<ll> a[m + 1];

    for (ll i = 1; i <= m; i++) {
        ll x;
        std::cin >> x;

        for (ll j = 1; j <= x; j++) {
            ll y;
            std::cin >> y;

            a[i].push_back(y);
        }
    }

    bool ans = true;

    for (ll i = 1; i <= m; i++) {
        for (ll j = i + 1; j <= m; j++) {
            std::vector<ll> val(n + 1);

            for (auto u : a[i]) {
                val[u]++;
            }

            for (auto u : a[j]) {
                val[u]++;
            }

            bool check = false;

            for (ll k = 1; k <= n; k++) {
                if (val[k] == 2) {
                    check = true;
                }
            }

            if (!check) {
                ans = false;
            }
        }
    }

    if (ans) {
        std::cout << "possible\n";
    } else {
        std::cout << "impossible\n";
    }
}
