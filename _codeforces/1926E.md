during each stage, the numbers eliminated are of the form (2k - 1), and on each successive stage, we multiply it by 2. so we first delete all numbers of the form (2k - 1), then (4k - 2) and then (8k - 4), etc.

so we just have to keep a track of count of numbers deleted in each stage and get the ith number using the formula for the stage.
```c++
void solve() {
    i64 n, k; cin >> n >> k;
    i64 i = 1, j = 2;
    while (k > 0) {
        i64 cur = (n + i) / j;
        if (k > cur) {
            k -= cur;
            i *= 2;
            j *= 2;
        } else {
            i64 l = k * j - i;
            cout << l << '\n';
            return;
        }
    }
}   
```
