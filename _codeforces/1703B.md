since first solve receives extra balloon, hence the number of balloons distributed is n + number of unique problems.

```c++
void solve() {
    i64 n; cin >> n;
    string s; cin >> s;
    set<char> t;
    for (auto& u: s) t.insert(u);
    cout << n + t.size() << '\n';
}
```
