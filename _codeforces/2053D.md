## Explanation

Let

$$
P = \prod_{i=1}^{n} \min(a_i,b_i).
$$

After every update, we are allowed to rearrange both arrays arbitrarily before computing the product. Therefore, the first question is how to maximize $P$ for fixed arrays.

**Lemma 1.** The maximum value of

$$
\prod_{i=1}^{n}\min(a_i,b_i)
$$

is obtained by sorting both arrays in non-decreasing order and pairing equal ranks together.

**Proof.**

Since multiplication is commutative, we may sort array $a$ without loss of generality.

Suppose we want to choose which element of $b$ is paired with the largest element $a_n$. Let $b_i$ be the largest unused element of $b$, and suppose an optimal solution instead pairs $a_n$ with some smaller element $b_j$.

If $b_i \le a_n$, then

$$
\min(a_n,b_i)=b_i \ge b_j=\min(a_n,b_j).
$$

If $b_i > a_n$, then

$$
\min(a_n,b_i)=a_n \ge \min(a_n,b_j).
$$

Hence replacing $b_j$ by $b_i$ never decreases the contribution of the last position. By an exchange argument, the largest remaining element of $b$ should always be paired with the largest remaining element of $a$.

Repeating this argument inductively proves that sorting both arrays and pairing equal ranks is optimal.

Therefore

$$
P
=

\prod_{i=1}^{n}
\min(A_i,B_i),
$$

where $A$ and $B$ denote the sorted versions of $a$ and $b$.

Computing this from scratch requires $\mathcal{O}(n \log n)$ time.

However, doing this after every update leads to an $\mathcal{O}(qn\log n)$ solution, which is too slow.

### Maintaining the Product

Suppose we have two operations:

1. Given an original position, determine its current value in the sorted array.
2. Given a value in one sorted array, determine the largest value paired with it in the other sorted array.

Let these operations be denoted by $F$ and $G$ respectively.

Consider an update that increments some value from $v$ to $v+1$ in array $a$.

Let

$$
B = G(v)
$$

be the value currently paired with it.

Only one factor of the product changes:

$$
\min(v,B)
\longrightarrow
\min(v+1,B).
$$

Hence

$$
P
=

P
\cdot
\frac{\min(v+1,B)}
{\min(v,B)}.
$$

So if we can maintain the sorted arrays dynamically, each update can be processed in constant time apart from maintaining the ordering information.

### Data Structure

Store:

* the original arrays `a` and `b`,
* sorted copies `A` and `B`,
* for every value, the first and last position where it occurs in the sorted arrays.

Specifically,

```text
A1[value] = first occurrence in A
A2[value] = last occurrence in A

B1[value] = first occurrence in B
B2[value] = last occurrence in B
```

When a value `v` is incremented:

* move the last occurrence of `v` into the block of value `v + 1`,
* update the first/last occurrence maps,
* modify exactly one factor of the product.

All updates therefore take $\mathcal{O}(\log n)$ due to map operations.

## Code

```cpp
void solve() {
    i64 n, q; cin >> n >> q;
    vector<i64> a(n + 1), b(n + 1);

    for (int i = 1; i <= n; i++) cin >> a[i];
    for (int i = 1; i <= n; i++) cin >> b[i];

    vector<i64> A = a;
    vector<i64> B = b;

    sort(A.begin() + 1, A.end());
    sort(B.begin() + 1, B.end());

    z2 ans = 1;
    for (int i = 1; i <= n; i++) {
        ans *= min(A[i], B[i]);
    }

    cout << ans << ' ';

    map<i64, i64> A1, A2, B1, B2;

    for (int i = 1; i <= n; i++) {
        if (A1.count(A[i])) {
            A2[A[i]] = i;
        } else {
            A1[A[i]] = i;
            A2[A[i]] = i;
        }

        if (B1.count(B[i])) {
            B2[B[i]] = i;
        } else {
            B1[B[i]] = i;
            B2[B[i]] = i;
        }
    }

    while (q--) {
        i64 o, x;
        cin >> o >> x;

        if (o == 1) {
            i64 val = a[x];
            a[x]++;

            i64 largest = B[A2[val]];

            ans *= min(val + 1, largest);
            ans /= min(val, largest);

            A1[val + 1] = A2[val];

            if (!A2.count(val + 1)) {
                A2[val + 1] = A1[val + 1];
            }

            A[A1[val + 1]] = val + 1;

            A2[val]--;

            if (A2[val] < A1[val]) {
                A1.erase(val);
                A2.erase(val);
            }
        } else {
            i64 val = b[x];
            b[x]++;

            i64 largest = A[B2[val]];

            ans *= min(val + 1, largest);
            ans /= min(val, largest);

            B1[val + 1] = B2[val];

            if (!B2.count(val + 1)) {
                B2[val + 1] = B1[val + 1];
            }

            B[B1[val + 1]] = val + 1;

            B2[val]--;

            if (B2[val] < B1[val]) {
                B1.erase(val);
                B2.erase(val);
            }
        }

        cout << ans << ' ';
    }

    cout << '\n';
}
```
