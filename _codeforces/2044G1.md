## Explanation

Observe that every vertex points to exactly one other vertex. Therefore the graph is a **functional graph**.

A standard property of functional graphs is that every connected component consists of:

- a directed cycle,
- with rooted trees directed towards the cycle.

The process described in the problem naturally suggests repeatedly removing leaves.

---

### Topological Trimming

**Definition.** A vertex is called a leaf if its indegree is zero.

Consider the following process:

1. Remove all current leaves.
2. Recompute indegrees.
3. Repeat until no leaves remain.

Let

$$
T
$$

denote the number of seconds required until this process can no longer delete any vertex.

---

### Lemma 1

After repeatedly deleting leaves, the only remaining vertices belong to directed cycles.

**Proof.**

Suppose a vertex does not belong to a cycle.

Since every vertex has outdegree exactly one, following outgoing edges from that vertex must eventually reach a cycle.

Hence every non-cycle vertex belongs to a tree directed toward some cycle.

Every finite rooted tree contains a leaf. Therefore, during the trimming process, all vertices of such trees are eventually removed layer by layer.

A cycle vertex can never become a leaf because it always has an incoming edge from another vertex of the cycle.

Therefore, after all possible deletions, precisely the cycle vertices remain.
$\square$

---

### Lemma 2

A vertex removed at second $t$ has distance exactly $t-1$ from the nearest cycle.

**Proof.**

Vertices with indegree zero initially are exactly the leaves of the trees attached to cycles. These are removed at time $1$.

After removing all vertices at distance $d$, the vertices at distance $d+1$ become the new leaves and are removed in the next second.

By induction, vertices at distance $d$ from the nearest cycle are removed at time

$$
d+1.
$$

Hence the last deleted vertex is removed after

$$
T
$$

seconds, where $T-1$ is the maximum distance from any vertex to a cycle.
$\square$

---

### Lemma 3

The answer equals

$$
T + 2.
$$

**Proof.**

After $T$ seconds, all tree vertices have been removed and only cycle vertices remain.

The remaining structure is therefore a directed cycle.

The problem requires two additional steps once all trees have been eliminated, corresponding to resolving the final cycle.

Hence the total number of seconds is

$$
T + 2.
$$

$\square$

---

### Algorithm

1. Construct the functional graph.
2. Compute indegrees.
3. Insert all vertices with indegree zero into a queue.
4. Perform Kahn's topological trimming process.
5. Let $T$ be the largest deletion time assigned to any vertex.
6. Output

$$
T + 2.
$$

### Complexity

Each vertex enters the queue at most once and each edge is processed exactly once.

Therefore the time complexity is

$$
\mathcal{O}(n)
$$

and the memory complexity is

$$
\mathcal{O}(n).
$$

## Code

```cpp
void solve() {
    i64 n;
    cin >> n;

    vector<i64> a(n + 1);
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
    }

    vector<vector<i64>> adj(n + 1);
    vector<i64> indeg(n + 1);

    for (int i = 1; i <= n; i++) {
        indeg[a[i]]++;
        adj[i].push_back(a[i]);
    }

    queue<array<i64, 2>> q;

    for (int i = 1; i <= n; i++) {
        if (indeg[i] == 0) {
            q.push({1, i});
        }
    }

    i64 time = 0;

    while (!q.empty()) {
        auto [t, f] = q.front();
        q.pop();

        time = max(time, t);

        for (auto& u : adj[f]) {
            indeg[u]--;

            if (indeg[u] == 0) {
                q.push({t + 1, u});
            }
        }
    }

    cout << time + 2 << '\n';
}
